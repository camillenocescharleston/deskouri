# Deskouri 🎫

> **Safe to Fail. Built to Learn.**

Deskouri is a free, open source IT practice platform for students worldwide. 
Practice real ticketing workflows, incident response, Active Directory labs, 
and security simulations — just like ServiceNow, Jira, and Zendesk, but free.

## 🌐 Live Platform
[deskouri.org](https://deskouri.org)

## ✨ Features
- 🎫 Ticketing workflows (Help Desk, ITIL-based)
- 🛡️ Incident response simulations (phishing, ransomware)
- 📡 Packet analysis labs (Network+, CySA+)
- 🪟 Active Directory labs (simulated + Azure AD)
- 🐙 GitHub integration — build your portfolio
- 🤖 AI agent builder
- 📊 Progress tracking & certificates

## 🎓 Cert-Aligned
CompTIA A+ · Network+ · Security+ · CySA+
